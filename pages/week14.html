Files

A file is a collection of data stored in a drive, a storage device. They are made up of 1s and 0s

The Os differentiates file types by their file extension, magic numbers and file metadata

File Extensions
It's the characters that are on the end of the file: .jpeg, .mov, .html
Magic Numbers
Magic numbers are the first few bytes of a file that contain the file signature
File Metadata
Files contain also data about themselves and that's called metadata

How does the computer store sound and light in 1s and 0s?

Audio files are encoded through converting sound into a continuous electrical signal which is then stored as a stream of binary data.
The digital data is often compressed using a codec like .WAV or .MP3

Image files, there are two types: Vector which use geometry to draw pictures and raster which use individual pixels with each a three 8-bits values, RGB, red, green, blue

File compression is an important aspect of files, there are two types:

Lossless
Keeps all original data and reduces it using Redundancy reduction, which basically replaces consecutive data with a single value and a count and
Dictionnary based, which replaces repeated patters with a code which is stored in a dictionnary.
Lossy
Permanently removes data that the human brain cannot comprehend,
for example in an audio file it will remove sounds that the human brain cannot hear, so what's the point of keeping it?
