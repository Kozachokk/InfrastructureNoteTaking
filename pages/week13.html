Networks

A typical network is multiple devices that communicate with one another. There are two types:

LAN(Local Area Network)
LAN is a network that is limited to one place, a small one might be your home, where your cellphones, TVs, laptops connect together by a router.
A larger one might be a school or work network, where there is typically multiple access points to the network through switches, hubs and wireless access points
WAN(Wide Area Network)
WAN is a network of LANs, where LANs communicate together. The whole internet is a WAN

Network Components:

Network Interface Card(NIC)
It's a piece of hardware that connect the device to the network
Transport Medium
It's how data is tranfered through, by cables, wirelessly
Hubs
Hubs receive data and send it to every computer in the network, it's isn't recommended to use since traffic colision are very likely
Bridges
Bridges connect one network to another, they have two ports, one for receiving data, one for sending, typically used in differenct network architectures
Switches
Switches is a combination of hubs and bridges, they have multiple ports and have an address for each device and when receiving a message,
forwards it towards the correct device address, they typically connect multiple devices in a network
Routers
Routers connect multiple networks together it's the hub for a LAN, they also provide security like firewalls

Peer to peer(P2P) connections is a type of network architecture where device share ressources by directly connecting to one another without going through a server
A client/server connection is when devices(clients) connect to one server to access information, where the information is centralized

Network Protocols are rules that direct how data should be transmitted

Types of network protocols

  • Communication Protocols: Automation, Instant Messaging, Routing, Bluetooth, File Transfer, Internet Protocol
  • Network Management Protocols: Connection, Link Aggregation, Troubleshooting
  • Security Protocols: Encryption, Entity Authentication, Transportation
  • Network Protocols: HTTP, HTTPS, SSH, SMS
  • Transport Protocols: TCP, IP

To transfer data most networks will use TCP and IP, IP is the Internet Protocol which obtains the address to which data is sent,
TCP which is Transport Control Protocol which breaks the file into smaller packets and sends it over the network and handles the delivery

IPv4 is the most widely used IP, it's in a 32-bit format: 255.255.255.255
The new generation IPv6, that's slowly phasing in, is in 128-bit format: FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF
