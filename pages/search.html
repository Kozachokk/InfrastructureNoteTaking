What is a search engine?

A search engine is a program or algorithm that searches a database of cached websites using user's prompts and search operators and shows the most relevant sites on top, it uses machine learning and AI to refine those searches

But how does it know which pages to show?

It uses "spiders" that go starts in a website and follows it's hyperlinks, the pages with the most hyperlinks going to them get ranked towards being first

What it a browser then?

A browser is program that reads html, css and javascript on a given webpage and displays it, as a rule of thumb they also include search engines to find those websites
