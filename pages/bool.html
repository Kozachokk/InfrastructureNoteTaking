What is boolean logic?

Boolean logic is logic with binary states, on = 1, off = 0
It is used to represent data in a computer and we can do complexe computation with boolean logic and logic gates

What are logic gates?

Logic gates are circuits created using transistors that control the flow of electricity based on their input. There are 4 types of logic gates:

  • OR: returns true if any one of inputs is true
  • AND: returns true if both inputs are true
  • NOT: reverses the input
  • XOR: returns true if only on input is true
