What is a CPU?

Central Processing Unit(CPU):
  CPUs are considered the brain of the computer since they are performing all the calculations. A CPU has three main components:

  • Control Unit(CU):
       Controls the flow of the information
  • Arithmetic Logic Unit(ALU):
       The component that performs the calculations
  • Registers:
       Stores data and instructions for and from the ALU


CPUs these days are multi-cored, meaning in one piece there are multiple CPU that each have Level 1 and Level 2 Cache, and a Level 3 Cache that is shared with all the CPUs.
In order to communicate with the rest of the computer the CPU connects with the motherboard with pins. Also these days, there are two types of CPUs sockets,
we have Land Grid Array, that mean the the CPU uses Lands and the motherboard has the pins, with a Pin Grid Array CPU it's vice-versa.


A typical CPU cycle is Fetch ==> Decode ==> Execute ==> Store, and modern CPUs run billions of those cycle per second
which is coincidentally also the measurement of Hz, so most modern CPUs are in the GHz.

  • Fetch: Fetches the instruction
  • Decode: Converts the instruction to machine code
  • Execute: Runs the translated machine code
  • Store: Stores the result in memory

Another important thing about the cycle is that CPUs can pipeline, meaning they can run those four operations at the same time
instead to running them one by one and starting over. With pipelining it's possible to run 4 commands with the CPU
