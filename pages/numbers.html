What is a numbering system?

A numbering system is a representation of numbers, there are infinite ways to represent quantity, but in this course the ones important are:

  • Base 10(Decimal): Most common system, used among us. 10 possible values: 0-9
  • Base 2(Binary): System used by computers to represents data and state: 2 possible values: 0-1
  • Base 16(Hexadecimal): System used by us to represent data in computers: 16 possible values: 0-F
  • Base 8(Octal): Used in Unix file permissions. 8 Possible values: 0-7

Binary is use to represent data and a group of 8 bits is a byte, with a total of 256 possible combinations

ASCII uses 7 bits to represent the whole alphabet, numbers and other special signs, with a total of 128 combinations

Extended ASCII uses one byte, so 128 combinations more than ASCII

Unicode(U+0000) consists of planes 17, with each their own code points
