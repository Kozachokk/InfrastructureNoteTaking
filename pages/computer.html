What is a computer?

A computer is an electronic device that accepts data, processes it and produces output
The computer generally has a motherboard, a cpu, a ram, some storage, plus you can get a gpu, some peripherals, etc.

  • motherboard: Center piece of a computer that connects the hardwares together
  • CPU: Central Processing Unit, it processes the data
  • RAM: Random Access Memory, it acts like temporary storage for running software and operating systems
  • SSD/HDD: Storage, stores data
  • GPU: Graphical Processing Unit, it processes data along side the cpu
  • Preripherals: Input/Output, monitors, keyboard, mouse, speaker, microphone, etc.

What is hardware/software?

Hardware is the physical parts of the computer, while software is the virtual. Hardware can be the cpu, storage, monitors, etc. Software can be applications like Word, Chrome, Minecraft, etc.
The operating system, like Windows, macOS, Linux, connects the hardware with the software.
